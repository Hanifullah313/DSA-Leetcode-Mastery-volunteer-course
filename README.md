# DSA + LeetCode Mastery 🚀

Welcome to the **DSA + LeetCode Mastery** repository! This course is a volunteer-led initiative designed to build a rock-solid foundation in Data Structures and Algorithms, transitioning from linear fundamentals to complex hierarchical structures.

This course is led by **Hanif Ullah** (Lead & Trainer), in collaboration with **Jaweria Siddique** and **Haris Khan** (Trainers).

---

## 📂 Repository Contents

* `code/` → Python code files for each class
* `recordings/` → Session recordings
* `assignments/` → Weekly practice assignments
* `resources/` → Extra reading materials & links
* `projects/` → Mini-projects & final project code

## 🎯 Learning Objectives

* Build a **solid foundation in Python** programming.
* Develop **problem-solving skills** through assignments & projects.
* Learn how Python is used in **real-world applications**.
* Gain confidence to advance into **data science, web dev, or automation**.

## 🗓️ Course Outline & Weekly Schedule

### **Week 1: Linear Fundamentals (Arrays, Hashing & Strings)**
* **Day 1:** Data Structure Basics & Complexity (Big-O notation).
* **Day 2:** Array Basics (Traversal & Accumulation).
* **Day 3:** Array Manipulation (In-place modification).
* **Day 4:** Hashing (Hash Map vs Hash Set).
* **Day 5:** Strings (Character traversal & Anagram concepts).

### **Week 2: Two Pointers & Sliding Window**
* **Day 1:** Opposite Direction Pointers.
* **Day 2:** Slow & Fast Pointers.
* **Day 3:** Two Pointers on Sorted Arrays.
* **Day 4:** Fixed Sliding Window.
* **Day 5:** Variable Sliding Window.

### **Week 3: Searching & Sorting**
* **Day 1:** Linear Search & Sequential Traversal.
* **Day 2:** Binary Search (Logarithmic searching).
* **Day 3:** Basic Sorting (Bubble, Selection, Insertion).
* **Day 4:** Sorting Applications.
* **Day 5:** Divide and Conquer (Merge Sort & Quick Sort).

### **Week 4: Linked Lists, Stacks & Queues**
* **Day 1:** Linked List Basics & Traversal.
* **Day 2:** Linked List Manipulation (Reversal/Deletion).
* **Day 3:** Stack (LIFO concepts).
* **Day 4:** Queue (FIFO concepts).
* **Day 5:** Deque (Sliding Window Maximum).

### **Week 5: Trees and Recursion**
* **Day 1:** Recursion Basics & Call Stacks.
* **Day 2:** Recursion Patterns (Decision Trees).
* **Day 3:** Tree Basics & Binary Trees.
* **Day 4:** DFS (Preorder, Inorder, Postorder).
* **Day 5:** BFS (Level Order Traversal).

### **Week 6: Backtracking & Graphs**
* **Day 1:** Backtracking Basics (Decision Tree).
* **Day 2:** Backtracking Variations (Permutations).
* **Day 3:** Backtracking on Grids (Word Search).
* **Day 4:** Graph Basics & BFS Pathfinding.
* **Day 5:** Graph DFS (Connected Components).
